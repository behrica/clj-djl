* clj-djl

A Clojure library wrap deep java learning(DJL.ai)

** Usage

clj-djl is available as a Maven artifact from Clojars.

With Leiningen/Boot:

#+begin_src clojure
[clj-djl "0.1.1"]
#+end_src

** Examples

Try examples with [[https://github.com/kimim/clj-djl-examples][clj-djl-examples]].
